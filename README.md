# Parallax-Website
This is a simple Parallax Website project made using HTML, CSS, and JavaScript. It uses a scrolling effect where the background moves slower than the content, creating a cool 3D feel. Built as a fun project to explore modern web design and improve front-end skills.
# 🌐 Parallax Website

A modern and visually appealing **Parallax Scrolling Website** built using HTML, CSS, and JavaScript. This project creates a smooth scrolling experience where background and foreground elements move at different speeds, giving a stunning 3D-like effect.

---

## 🚀 Features

- ✨ Smooth parallax scrolling effect  
- 🎨 Clean and responsive design  
- ⚡ Lightweight and fast  
- 📱 Mobile-friendly layout  
- 💡 Beginner-friendly project  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript
